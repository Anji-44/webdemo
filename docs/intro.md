# Docs-only websites

Welcome to the world of **docs-only websites** - where documentation becomes the primary product, not just an afterthought.

## What Are Docs-Only Websites?

Docs-only websites are purpose-built platforms where documentation takes center stage. Unlike traditional websites that treat docs as supplementary, docs-only sites put your content front and center, creating focused, reader-centric experiences.

## The Power of Docs-First Approach

**Why docs-only websites matter:**

- **Focused Experience**: Readers get exactly what they need - clear, structured information without distractions
- **Better Navigation**: Designed specifically for how people consume technical content
- **Search-Optimized**: Built from the ground up for findability and discoverability
- **Mobile-First**: Documentation that works seamlessly across all devices

## When to Choose Docs-Only

Perfect for:
- API documentation and developer guides
- Product manuals and user guides
- Knowledge bases and support documentation
- Technical tutorials and learning resources
- Open source project documentation
